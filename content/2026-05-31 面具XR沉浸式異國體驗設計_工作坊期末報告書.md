# 面具 × XR 沉浸式異國體驗設計：參與式設計工作坊期末報告書

---

## 壹、專案概述（Project Overview）

### 1.1 研究背景與動機

現代都市生活節奏快速、壓力繁重，許多人渴望出國旅行以獲得紓壓與文化體驗，卻因經濟條件、時間限制、健康狀況或社會責任等現實因素而無法成行。這種「渴望卻無法實現」的心理落差，反而加深了日常生活中的焦慮與倦怠感。

面具，表面上是遮蔽面容的工具，實質上卻是一面映照人類文明的鏡子。從義大利威尼斯模糊社會階級的華麗面具，到非洲部落中溝通神靈的儀式面具，再到日本能劇中神靈與傳統藝術的化身——同一種形式在不同地域被賦予了完全不同的生命與功能。面具承載了成百上千年的集體記憶，是跨越時空的文化載體。

本專案以「面具」作為身分轉換的入口，結合延展實境（XR）技術，探索如何為無法出國的都市人設計一段「戴上面具即可沉浸體驗異國風情」的療癒空間與服務流程。透過參與式設計工作坊的方法，從使用者的真實需求出發，系統性地建構一套完整的沉浸式體驗設計。

### 1.2 專案目標

本專案旨在透過「參與式設計工作坊」，運用服務設計的雙鑽石模型，開發出一套結合「面具文化意涵」與「XR 沉浸療癒空間服務時間軸」的沉浸式異國體驗。具體目標包含：

1. **理解使用者**：透過人物誌與觀察分析，深入理解渴望出國卻無法成行之使用者的生活情境、痛點與潛在需求。
2. **發散與收斂**：運用研究牆、五感體驗想像與親和圖分群，從大量感官素材中篩選出最具療癒潛力的設計元素。
3. **設計旅程**：建構完整的顧客旅程圖，涵蓋從「進入前」到「離開」的五個階段，並注入使用者的主觀情緒弧線。
4. **產出原型**：利用 AI 圖像生成工具，快速視覺化面具外型與場域情境，作為後續開發的概念驗證。

### 1.3 預期效益

- **對沉浸式體驗設計領域**：提供一套可複製的「文化載體 × XR 技術 × 服務設計」整合方法論，示範如何將抽象的文化體驗轉化為可設計的服務觸點。
- **對潛在消費者**：為無法出國的都市人提供一種新型態的療癒紓壓體驗選擇，透過多感官沉浸設計達到情緒轉換與壓力釋放。
- **對 XR 產業與文化體驗業者**：開拓「面具 × XR × 文化體驗」的創新商業模式可能性，結合在地文化資源與數位技術創造差異化服務。

## 貳、研究與設計方法論（Methodology）

### 2.1 雙鑽石模型（Double Diamond Model）

本工作坊採用英國設計委員會（Design Council）提出的雙鑽石模型作為核心設計框架，透過四個階段的「發散—收斂—再發散—再收斂」節奏，引導參與者從模糊的問題空間走向具體的解決方案：

- **探索需求（Discover）**：透過人物誌閱讀、觀察便利貼、五感體驗想像與研究牆聯想發散，打開對使用者需求與文化感官素材的探索範圍。
- **定義問題（Define）**：透過親和圖分群、療癒元素票選與 HMW 問題框架，將大量發散素材收斂為明確的設計方向與挑戰。
- **產品方案（Develop）**：透過顧客旅程圖建構與情緒弧線設計，將篩選出的療癒元素轉化為具體的服務流程與體驗設計。
- **解決方案（Deliver）**：透過 AI 圖像生成面具與場域原型，將設計概念視覺化，並經由成果發表進行跨組回饋與驗證。

### 2.2 參與式設計與共創（Co-creation）

本工作坊將 9 位參與者分為 3 組（每組 3 人），每組對應一個國家/文化（日本、威尼斯、非洲），從各自的文化脈絡出發進行共創設計。參與式設計的核心價值在於：

- 打破單一設計師的假設，將使用者的真實經驗與想像納入設計過程。
- 透過跨背景的參與者碰撞，產生個人視角無法企及的創意火花。
- 讓每個步驟的產出成為下一步的輸入，形成環環相扣的設計鏈條。

### 2.3 工作坊原則

- **先觀察，再發想**：不急於跳入解決方案，而是先透過人物誌深入理解使用者。
- **發散不批判**：在探索與發想階段，鼓勵「先寫再說」，不過早評判可行性。
- **動手做，不只用嘴說（Doing, not talking）**：透過便利貼書寫、研究牆建構、旅程圖繪製與 AI 原型生成，讓設計思考以「動手做」的方式落地。
- **每步有產出，步步有銜接**：每個階段的產出都是下一步的輸入，參與者不需猜測接下來要做什麼。

## 參、第一階段：探索需求（Phase 1: Discover）

### 3.1 開場暖身與情境導入

#### Slido 暖身互動

工作坊開場以 Slido 即時互動工具引導全班進入主題情境：「你有沒有出過國？有沒有什麼特別難忘的體驗？或者，你最想去什麼樣的國家，想體驗什麼？」螢幕投出回覆雲，講師快速掃過有趣的回覆帶動討論氛圍，並引出核心設計問題——「如果有一個人渴望出國但無法出去，我們要怎麼替他設計一個『戴上面具就能沉浸體驗』的空間？」

#### 面具文化背景簡介

簡述威尼斯、非洲、日本三種文化中面具的意義：威尼斯面具模糊社會階級、非洲面具溝通神靈世界、日本能劇面具承載神靈與藝術的化身。帶出核心概念：「面具 = 身分轉換的入口、跨越時空的文化載體」。

### 3.2 人物誌閱讀與脈絡觀察

#### 人物誌閱讀

每組領取一份對應特定國家/文化的虛構人物誌資料，包含：日常作息、生活環境、人際互動、關鍵物件與人物特質五個面向。例如日本組的人物誌描述了一位 25 歲東京上班族小林美咲——朝九晚八的辦公室工作、窗外高架橋的小套房、牆上貼著京都嵐山海報、隨身攜帶三年前在伏見稻荷買的御守。

組員先安靜閱讀 3 分鐘各自吸收，不急於討論。

#### 觀察紀錄便利貼

閱讀完成後，每人獨立撰寫至少 3 張觀察便利貼，格式為「觀察到的具體細節 → 可能代表的含義」。重點在於從人物誌中「觀察到的具體細節」而非「感想」，包含推敲出的經濟狀況、心理狀態、作息規律等。

> **便利貼範例**：
> - 「她的房間貼了京都嵐山的海報 → 可能特別嚮往嵐山的竹林和安靜感」
> - 「她通勤時用 AirPods → 聽覺對她的日常很重要，可能也用音樂逃離通勤噪音」
> - 「御守是三年前買的，現在還留著 → 對京都有強烈的情感依附，想回去或想延續那段體驗」

全部貼到白板觀察區後，組員快速口頭分享各自觀察。

- **預期產出**：每組白板上至少 9 張「觀察細節→含義」便利貼。

### 3.3 五感體驗想像與研究牆建立

#### 五感體驗想像

白板上預設五個感官區域（👁 視覺 / 👂 聽覺 / 👃 嗅覺 / ✋ 觸覺 / 👅 味覺），每人各在每個感官下寫便利貼，站在人物誌使用者的角度想像療癒放鬆體驗的感官期待。完成後，小組投票或口頭討論選出最有共識的 5 個體驗（每感官各 1 個）。

> **範例便利貼**：
> - 👁 視覺：竹林的光影、嵐山的薄霧晨光、京都的木造建築
> - 👂 聽覺：竹林裡的風聲、寺廟的鐘聲、遠處的流水聲
> - 👃 嗅覺：線香的氣味、新鮮茶葉的清香、雨後泥土味
> - ✋ 觸覺：木製地板的質感、茶杯的溫熱、石板路的粗糙感
> - 👅 味覺：抹茶的苦甜、熱湯的溫暖感

#### 研究牆聯想發散

以國家為核心、五感共識為第一層，進行放射狀自由聯想。從每張共識便利貼往外延伸關聯詞彙、元素、畫面或情境，持續往第二層、第三層拓展。聯想不限五感，任何與國家有關的元素都可以：食物、傳說、氣候、時節、建築材質、歷史片段、當地人的日常等。

- **預期產出**：
  - 每組一份「五感體驗共識清單」（5 個體驗）。
  - 每組一張以「國家」為核心的放射狀發散研究牆（至少 30 張便利貼）。

## 肆、第二階段：定義問題（Phase 2: Define）

### 4.1 便利貼分群與親和圖（Affinity Clustering）

將研究牆上的便利貼全部複製到「分群區」，三人共同討論進行分群（意思相近的放在一起）。每個群組不超過 8 張便利貼——超過代表分類太籠統，需再細分。每個群組貼上命名便利貼作為群組標題，命名要精準（例如「靜謐自然空間」而非「自然」）。

- **預期產出**：整理完成的分群白板，4-8 個命名群組。

### 4.2 療癒元素票選（Dot Voting）

每人 5 票，投給最有設計潛力、最能呼應使用者需求的項目。可以全押一個群組，也可以分散。鼓勵把票投給「看起來最難但最有感的」，而不是「最好做的」。得票 ≥ 1 的項目整理成一份「療癒元素設計清單」，貼在白板顯眼位置。

- **預期產出**：一份票選後的「療癒元素設計清單」。

### 4.3 框架問題（How Might We, HMW）

基於人物誌的痛點與療癒元素清單，將限制與需求轉化為設計挑戰問句，融入旅程圖的各階段設計中。

> **HMW 問題範例**：
> - 「我們可以如何讓使用者在戴上面具的瞬間，就感覺自己身處京都的竹林小徑？」
> - 「我們可以如何讓出戲的過程不是突然被拉回現實，而是被溫柔地送回來？」
> - 「我們可以如何在離開時，讓使用者帶走一個能延續今天體驗記憶的觸發物？」

- **預期產出**：優先排序的 HMW 設計挑戰清單。

## 伍、第三階段：產品方案（Phase 3: Develop）

### 5.1 顧客旅程圖建構與體驗元素置入

以票選出的療癒元素清單為素材，在旅程圖模板上設定五個階段，將療癒元素分配到各階段，明確標示使用者行為/動作、空間設計重點、體驗元素及具體呈現方式。

| 旅程階段               | 使用者行為/動作   | 空間設計重點       | 體驗元素（從清單中選） | 呈現與應用方式   |
| ---------------------- | ----------------- | ------------------ | ---------------------- | ---------------- |
| 進入前（到達空間外）    |                   |                    |                        |                  |
| 入戲準備（走進空間）    |                   |                    |                        |                  |
| XR 沉浸體驗            |                   |                    |                        |                  |
| 出戲緩衝               |                   |                    |                        |                  |
| 離開（帶走紀念）        |                   |                    |                        |                  |

**日本組旅程圖填寫範例**：

| 旅程階段    | 使用者行為               | 空間設計重點            | 體驗元素     | 呈現方式                                  |
| ----------- | ----------------------- | ---------------------- | ------------ | ----------------------------------------- |
| 進入前      | 看到空間外觀，決定進入    | 和風木格子外觀，暖光引路燈 | 視覺：木建築  | 入口外設置紙燈籠，低調溫暖的燈光              |
| 入戲準備    | 換上和服腰帶布／換鞋      | 備品區、換鞋台、靜候廊道  | 嗅覺：線香    | 廊道播放低頻鐘聲，香氛機釋放線香氣味           |
| XR 沉浸體驗 | 戴上面具，進入虛擬嵐山竹林 | 坐/站的沉浸區，風扇裝置   | 聽覺：竹風聲  | 環境音播放竹葉沙沙與遠處流水，側吹微風扇        |
| 出戲緩衝    | 慢慢摘下面具，坐在靜區休息 | 柔光休息角，低矮木椅      | 觸覺：木質感  | 提供一杯熱煎茶，手握陶杯的溫熱觸感              |
| 離開        | 選取紀念品，緩步走向出口   | 紀念品小架，出口燈光漸亮  | 嗅覺：茶香    | 帶走一個小香囊或一張手寫和風卡，掃碼聽體驗音檔   |

- **預期產出**：每組一張填寫完整的顧客旅程圖（含體驗元素配置與呈現方式）。

### 5.2 主觀感受與情緒弧線設計

在旅程圖下方增加「使用者主觀感受（第一人稱書寫）」與「情緒標記」兩列，以使用者的內心視角撰寫主觀體驗感受。

| 旅程階段    | 使用者主觀感受（第一人稱書寫）                                       | 情緒標記                          |
| ----------- | ------------------------------------------------------------------ | --------------------------------- |
| 進入前      | 「我不太確定裡面是什麼，但那個燈光讓我想走進去看看……」                  | 😐 → 😌 微微期待                   |
| 入戲準備    | 「換上布的瞬間感覺好像換了一個身分，那個香味……有點像三年前在京都……」      | 😌 放鬆，有點懷舊                   |
| XR 沉浸體驗 | 「眼前突然出現竹林，我聽到風，感覺有點不真實但是……很想就這樣站著不動。」   | 😌 → 🥹 沉浸、有點想哭的感覺         |
| 出戲緩衝    | 「摘下面具有點捨不得，但那杯熱茶讓我慢慢回來，像是被溫柔地送回現實。」    | 😌 → 😊 平靜、溫暖                  |
| 離開        | 「我帶走了一個香囊。這個味道會不會讓我以後每次聞到都想起今天這個下午。」    | 😊 → 🥹 感動，有點不捨               |

以 emoji 或文字標記情緒狀態，並在旅程圖下方畫出一條折線的「情緒弧線」。情緒弧線不一定一路向上，而是有「期待→驚奇→沉浸→不捨→溫暖」的完整起伏節奏。

- **預期產出**：含主觀感受 + 情緒弧線的完整顧客旅程圖。

### 5.3 體驗空間與服務流程發想

#### 體驗空間分區概念

- **文化過渡區**：使用者從日常世界進入體驗空間的轉換區域，透過視覺與嗅覺的環境變化啟動身分轉換。
- **XR 沉浸主場域**：使用者穿戴面具（XR 載體）進入虛擬異國場景的核心體驗區域。
- **休憩回歸區**：使用者從沉浸體驗中緩步回歸現實的緩衝區域，提供溫飲與靜謐空間。

#### 服務互動腳本設計

為旅程各節點規劃互動腳本，確保每個觸點都有明確的感官體驗設計：入場時遞上文化意象布巾、XR 體驗中面具感應連動環境變化、出戲時提供溫飲與紀念物件的儀式感安排。

## 陸、第四階段：解決方案（Phase 4: Deliver）

### 6.1 AI 生成面具與場域原型（AI-assisted Concept Visualization）

三人分工進行原型產出：

| 角色           | 負責內容                                                               |
| -------------- | ---------------------------------------------------------------------- |
| **A（面具）**  | 根據旅程的文化主題，撰寫並輸入面具外型的 AI 提示詞，生成面具原型圖。       |
| **B（場域）**  | 根據旅程圖中 XR 沉浸體驗的場景描述，撰寫並輸入場域情境的 AI 提示詞，生成場域氛圍圖。 |
| **C（準備發表）** | 整理旅程圖重點，準備 2 分鐘的發表內容（使用者是誰→旅程設計亮點→圖像展示）。 |

#### 面具提示詞模板

```
請生成一張面具設計圖。
文化背景：[例如：日本能劇／威尼斯嘉年華／西非部落]
面具材質：[例如：塗漆木料、黃金鑲嵌、植物纖維編製]
色調：[例如：暗紅與金色、象牙白與墨藍]
設計特徵：[例如：有翅膀延伸、眉毛誇張上揚、面頰有花紋彩繪]
氛圍：[例如：神聖莊嚴、歡樂神秘、自然原始]
視角：正面，純色背景，博物館展示風格。
```

#### 場域情境提示詞模板

```
請生成一張室內空間的情境圖。
空間類型：[例如：XR 沉浸體驗空間、療癒冥想室]
文化元素：[例如：竹林光影投影、威尼斯水道壁畫、非洲鼓形陳設]
燈光氛圍：[例如：暖黃燭光、藍紫色星空投影、自然光從天窗灑落]
地板與材質：[例如：原木地板、石板路感、沙地]
氛圍：[例如：安靜神聖、奇幻夢境、部落圍火感]
視角：人視角，站在空間中央向前望。
```

- **預期產出**：每組 1 張 AI 生成的面具原型圖 + 1 張 AI 生成的場域情境圖。

### 6.2 成果發表與跨組回饋

各組發表結構：

1. 「我們的使用者是___，他最渴望的是___。」（15 秒）
2. 「我們設計的體驗旅程是這樣的——進來時___，XR 裡面看到/感受到___，離開時帶走___。」（1 分 30 秒）
3. 展示面具原型圖和場域情境圖，說明設計想法。（1 分鐘）

每組 3 分鐘，講師嚴格控時。透過三組不同文化（日本、威尼斯、非洲）的發表，對比不同文化意象在沉浸空間中的呈現差異。
